Premium Solution

Image Widget

Here we have an example of our "Premium Solution" image widget. This widget begins as an img tag on the page and features an optional hover action (enabled by default) that shows the widget's Product Name, Brand Name, List Price , and Sale Price (if applicable).

Features
  • Hover action touch-point
  • Parent-page product lightbox
  • Parent-page universal shopping cart
  • Parent-page checkout experience
Options
  • Hover action enabled/disabled
  • Custom image
  • Custom dimensions

The HTML - Sample Code HERE


        <img class="nopin"
        src="https://cloudinary-a.akamaihd.net/shopbeam/image/fetch/
        w_[width],h_[height],c_[crop mode]/[url-encoded image url]"
        width="[width]" height="[height]" alt="[brand name] - [product name]"
        id="shopbeam-widget-image-placeholder-[random uuid]"
        data-shopbeam-url="[shopbeam product url]
        &image=[default lightbox image index]
        &apiKey=[your shopbeam API key]"/>

Text Widget

Here we have an example of our "Premium Solution" text widget. This widget begins as an a ("anchor") tag on the page.

Hello there,
I'm a skirt!

Features
  • Make any HTML shoppable!
  • Parent-page product lightbox
  • Parent-page universal shopping cart
  • Parent-page checkout experience
Options
  • None

The HTML - Sample Code HERE


        <a data-shopbeam-url="[shopbeam product url]
        &image=[default lightbox image index]&apiKey=[your shopbeam apikey]"
        href="javascript:void(0)" title="[brand name] - [product name]"
        id="shopbeam-widget-text-link-[random uuid]">

Scale Solution

Image Widget

This is an example of our "Scale Solution" image widget. This widget features a different design than the "Premium" image widget. Similar to the "Premium" image widget, this widget begins as an img tag on the page and and also features an optional hover action (enabled by default) that shows the widget's Product Name, Brand Name, List Price , and Sale Price (if applicable); however, instead of opening a lightbox which takes over the page, this widget stays within it's iframe and provides an entire shopping experience right in place.

This means this widget can be used in ad creatives without publisher supoprt! (i.e. no publisher-side javascript / iframe-buster, etc.)

Features
  • Hover action touch-point
  • In-unit product discovery
  • In-unit universal shopping cart
  • In-unit checkout experience
Options
  • Hover action enabled/disabled
  • Custom image
  • Custom dimensions

The HTML - Sample Code HERE


        <img class="nopin"
        src="https://cloudinary-a.akamaihd.net/shopbeam/image/fetch/
        w_[width],h_[height],c_[crop mode]/[url-encoded image url]"
        width="[width]" height="[height]" alt="[brand name] - [product name]"
        id="shopbeam-in-frame-widget-image-placeholder-[random uuid]"
        data-shopbeam-url="[shopbeam product url]
        &image=[default lightbox image index]
        &apiKey=[your shopbeam API key]"/>
